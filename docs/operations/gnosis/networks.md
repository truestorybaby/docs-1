# Networks

Chainstack supports joining the following Gnosis Chain networks:

* Mainnet — public Gnosis Chain mainnet.
* Sokol testnet — public Gnosis Chain testnet.
* Chiado testnet — a proof-of-stake public Gnosis Chain testnet.
